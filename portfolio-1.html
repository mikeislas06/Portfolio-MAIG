El Patrón Churrasquería Brasileña

Description

  • El Patrón

This project involves the design and development of a website for a Brazilian churrascaria, aiming to promote the restaurant and provide relevant information to visitors. The project utilizes modern technologies such as React, useState, useEffect, SCSS, and various popular libraries like react-router-dom and react-icons. Furthermore, the goal is to create a visually appealing and attractive style to enhance the developer's GitHub profile.

Technology

  • HTML5
  • CSS3
  • React
  • SASS
